Frequently Asked Questions

Q: What if my scan finds nothing?

Great news! You'll still receive a report confirming no quantum-vulnerable algorithms were detected. This is valuable documentation for compliance purposes.

Q: Can you scan private repositories?

Not yet. Phase 1 only supports public GitHub repos. Private repo support (via read-only GitHub App) is planned for Phase 2.

Q: Do you store my code?

No. Code is cloned to ephemeral storage, scanned, and immediately deleted. We don't retain copies.

Q: What languages are supported?

Python, JavaScript, TypeScript, Java, Go, Rust, C, C++, C#, Ruby, PHP, Swift, Kotlin, plus config files (YAML, JSON, ENV, INI, TOML) and certificates (PEM, CRT, DER).

Q: What if I disagree with a finding?

Reports identify patterns that may indicate vulnerabilities. Some findings (like SHA-256 in HMAC contexts) may be acceptable. Use the report as a starting point for deeper analysis.

Q: How is payment handled?

ETH is locked in Moltlaunch escrow when you submit a request. After delivery, you have 24 hours to review. Payment auto-releases after the review period (and burns $QSHIELD supply).
