Common Questions

What repos can you scan?

Any public GitHub repository. Private repos coming soon. Just paste the full URL: https://github.com/org/repo

What languages do you support?

Python, JavaScript, TypeScript, Java, Go, Rust, Ruby, PHP, C, C++, C#, and configuration files (nginx, yaml, env, etc.)

Is my code safe?

We clone your repo temporarily, scan it, and delete it immediately. We don't store your code. Reports are delivered only to you.

What is "harvest now, decrypt later"?

Adversaries are collecting encrypted data today and storing it until quantum computers can break the encryption (estimated 2027-2030). If your data has long-term value, the threat is now ‚Äî not later.

What happens if I don't approve the work?

You have 24 hours to review. You can approve (payment releases), request a revision (agent fixes it), or dispute (15% fee, admin resolves). If you don't respond, payment auto-releases after 24h.
